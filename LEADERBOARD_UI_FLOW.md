## Score Calculation Formula

```
Base Score = (Waves Ã— 100) + (Zombies Ã— 10) + (Plants Ã— 5) + (Sun Ã· 10)

If Win:
  Score += 1000
  
  If PlayTime < 300 seconds (5 min):
    Score += 500
  Else If PlayTime < 600 seconds (10 min):
    Score += 300

Final Score = Base Score + Win Bonus + Speed Bonus
```
